# node lv5

내배캠 node lv5 프로젝트

API 명세

회원가입 /signup POST  
로그인 /login POST  

게시글 작성 /posts  POST  
게시글 조회 /posts  GET  
게시글 수정 /posts/:postId  PUT  
게시글 삭제 /posts/:postId  DELETE  

댓글 작성 /cmts/:postId  POST  
댓글 조회 /cmts/:postId  GET  
댓글 수정 /cmts/:postId/:commentId  PUT  
댓글 삭제 /cmts/:postId/:commentId  DELETE
